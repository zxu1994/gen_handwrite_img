# 汉字手写截图工具（本地网页）

## 使用方式

1. 在 Finder 中打开本项目文件夹，双击 `index.html`，使用浏览器（建议 Safari / Chrome）打开。
2. 在页面中间的米字格区域，用触摸板或鼠标按住并拖动书写汉字：
   - 先在起笔位置点击按下（或触摸），
   - 不松开手指，跟随触摸板移动即可绘制笔画。
3. 写完后点击【保存 PNG】按钮，会在浏览器中自动下载一张 **白底黑字** 的 PNG 图片（文件名中带时间戳）。
4. 若需要重新书写，点击【清除】按钮即可清空当前米字格内容。

## 说明

- 显示用的画布带有米字格背景，方便练字参考。
- 实际导出的 PNG 图片中只保留白色背景和黑色笔画，不包含米字格线。
- 该工具为纯前端静态网页，无需安装任何依赖或后端服务。
